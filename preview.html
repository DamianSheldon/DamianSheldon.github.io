iOS Core Animation 笔记

2014-09-03技术

本文记录 Core Animation 的学习笔记，涵盖 CALayer、隐式动画 与 显式动画 的核心概念。修道如逆水行舟，不进则退。代码即修行，迭代即飞升。每一行代码都是向道之路上的一个脚印。

CALayer 基础

每个 UIView 都有一个与之关联的 CALayer，负责视图的可视化内容渲染。开发者可以直接操作 layer 的 cornerRadius、shadowColor 等属性来实现丰富的视觉效果。

修道如逆水行舟，不进则退。代码即修行，迭代即飞升。

代码示例

let layer = view.layer
layer.cornerRadius = 8
layer.shadowColor = UIColor.black.cgColor
layer.shadowOpacity = 0.3
layer.shadowOffset = CGSize(width: 0, height: 2)
  • 隐式动画：修改 layer 属性时自动触发的默认动画
  • 显式动画：通过 CABasicAnimation 手动创建
  • 关键帧动画：CAKeyframeAnimation 实现复杂路径
概念描述
CALayer视图的底层渲染对象
CABasicAnimation单一属性的基础动画
